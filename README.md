# IMU_PROXIMITY
Arduino sketch for IMU (MPU6050) and proximity sensors (VCNL4010)

## Harware
Arduino uno
MPU6050 IMU Sensor(6 Dof)
Adafruit VCNL4010 IR Proximity Sensor
